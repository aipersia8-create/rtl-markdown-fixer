# RTL Markdown Fixer

ابزار چت‌باتی برای نمایش تمیز متن فارسی راست‌به‌چپ همراه با Markdown، کد، لینک، ایمیل و فرمول LaTeX.

## Features

- Chat-style Persian RTL Markdown fixer
- Safe LTR rendering for code, links, emails, paths and formulas
- Markdown and LaTeX rendering with copy actions
- Local conversation history drawer
- Search, rename, pin and delete conversations
- Animated creative AI/code/math background
- Dark modern UI

## Deploy

This project is ready for Vercel.

- Framework: Next.js
- Build command: `npm run build`
- Environment variables: none

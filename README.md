# RTL Markdown Fixer

ابزار وب برای نمایش تمیز متن فارسی راست‌به‌چپ همراه با Markdown، کد، لینک، ایمیل و فرمول LaTeX.

## Features

- Persian RTL Markdown preview
- Safe LTR rendering for code, links, emails, file paths and formulas
- LaTeX support with KaTeX
- Local history drawer with search, rename, pin and delete
- Animated tech / AI style 3D background
- Dark modern UI

## Deploy

This project is ready for Vercel.

- Framework: Next.js
- Build command: `npm run build`
- Environment variables: none
